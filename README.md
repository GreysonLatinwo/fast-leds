Assumptions:
    Running on a Raspberrypi
        Tested on Raspberry pi 4 8gb
    Leds strip
        ws2812
        63 pixels
        connected to pin 18
    pulseaudio installed


Useage:
    run ./build.sh
    run ./run.sh
    Connect to RPi via Bluetooth and play music.
        Makesure you play the audio through the hdmi output
    
    Webserver running on port 9000
